# Análisis de la Participación Política en Jóvenes

Este proyecto tiene como objetivo analizar la relación entre la participación política en jóvenes y diversos factores que pueden influir en ella, como la confianza en las instituciones, el interés en la política, el uso de redes sociales, la edad y el nivel educativo.

## Metodología
Para ello, se diseñó y aplicó una encuesta con preguntas estructuradas en formato cuantitativo, lo que permitió recopilar datos medibles y analizables. 

## Procesamiento de Datos
Posteriormente, los datos fueron procesados en *R*, donde se realizaron los siguientes procesos:
* Construcción de un data frame.
* Análisis estadísticos descriptivos.
* Creación de una matriz de correlación.
* Gráfica de dispersión.
* Modelo de regresión lineal.

## Conclusiones
Los resultados obtenidos permiten identificar qué variables tienen mayor influencia en la participación política, destacando especialmente el papel del interés en la política como factor determinante. Este ejercicio demuestra la utilidad de las herramientas cuantitativas para el análisis de fenómenos sociales y contribuye a una mejor comprensión del comportamiento político en jóvenes.
